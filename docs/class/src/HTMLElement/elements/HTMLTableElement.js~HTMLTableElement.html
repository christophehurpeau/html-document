    public get caption: HTMLTableCaptionElement | null source

    Is an <span><a href="class/src/HTMLElement/elements/HTMLTableCaptionElement.js~HTMLTableCaptionElement.html">HTMLTableCaptionElement</a></span> representing the first <caption> that is a child of the element, or null if none is found. When set, if the object doesn't represent a <caption>, a <span><a href="class/src/DOMException.js~DOMException.html">DOMException</a></span> with the HierarchyRequestError name is thrown. If a correct object is given, it is inserted in the tree as the first child of this element and the first <caption> that is a child of this element is removed from the tree, if any.

    public get rows: HTMLElement[] source

    Returns a live <span>HTMLCollection</span> containing all the rows of the element, that is all <tr> that are a child of the element, or a child or one of its <thead>, <tbody> and <tfoot> children. The rows members of a <thead> appear first, in tree order, and those members of a <tbody> last, also in tree order. The HTMLCollection is live and is automatically updated when the HTMLTableElement changes.

    public get tBodies: HTMLElement[] source

    Returns a live HTMLCollection containing all the <tbody> of the element. The HTMLCollection is live and is automatically updated when the HTMLTableElement changes.

    public get tFoot: HTMLElement | null source

    Is an <span><a href="class/src/HTMLElement/elements/HTMLTableSectionElement.js~HTMLTableSectionElement.html">HTMLTableSectionElement</a></span> representing the first <tfoot> that is a child of the element, or null if none is found. When set, if the object doesn't represent a <tfoot>, a <span><a href="class/src/DOMException.js~DOMException.html">DOMException</a></span> with the HierarchyRequestError name is thrown. If a correct object is given, it is inserted in the tree immediately before the first element that is neither a <caption>, a <colgroup>, nor a <thead>, or as the last child if there is no such element, and the first <tfoot> that is a child of this element is removed from the tree, if any.

    public insertRow(index: number): HTMLElement source

    Returns an <span><a href="class/src/HTMLElement.js~HTMLElement.html">HTMLElement</a></span> representing a new row of the table. It inserts it in the rows collection immediately before the <tr> element at the given index position. If necessary a <tbody> is created. If the index is -1, the new row is appended to the collection. If the index is smaller than -1 or greater than the number of rows in the collection, a <span><a href="class/src/DOMException.js~DOMException.html">DOMException</a></span> with the value IndexSizeError is raised.
